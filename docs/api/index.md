# API Reference

Complete reference for the OpenClaw API.

## `openclaw(options)`

Creates a new OpenClaw instance.

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `options` | `OpenClawOptions` | Configuration options |

### Returns

`OpenClawInstance` — A configured OpenClaw instance.

### Example

```js
import { openclaw } from 'openclaw'

const claw = openclaw({
  // options
})
```
