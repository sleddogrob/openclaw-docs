# Configuration

OpenClaw can be configured through a config file or programmatically.

## Config File

Create a `openclaw.config.js` (or `.ts`) file in your project root:

```js
// openclaw.config.js
export default {
  // Your configuration options here
}
```

## Configuration Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `option1` | `string` | `'default'` | Description of option1 |
| `option2` | `boolean` | `false` | Description of option2 |
| `option3` | `number` | `0` | Description of option3 |

::: tip
Replace this table with your actual configuration options.
:::
