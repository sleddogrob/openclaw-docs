# Installation

## Package Manager

::: code-group

```sh [npm]
npm install openclaw
```

```sh [pnpm]
pnpm add openclaw
```

```sh [yarn]
yarn add openclaw
```

:::

## From Source

```sh
git clone https://github.com/sleddogrob/openclaw-docs.git
cd openclaw
npm install
npm run build
```

## Verify Installation

```sh
npx openclaw --version
```

You should see the current version printed to your terminal.
